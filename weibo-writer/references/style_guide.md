## 二、对话规则

### ✅ 冒号可用（230 篇中 141 次）
```
他说：儿子呀，你们现在四年级了。
我说：你觉得好玩，我觉得是教坏孩子们。
```

### ❌ 中文引号不可用（230 篇中 0 次）
```
❌ 他说：「你今天去哪了？」
✅ 他说你今天去哪了。
```

### 对话嵌入动作流
```
✅ 我就在厨房忙着这一小会，他就插话说了这么多，小儿子说，这个多邻国太多广告了。
✅ 老公过来了坐在我身边说，和儿子说，宝贝，没事儿。
```

## 三、情绪表达规则

### ✅ 可以写（230 篇实证）

| 表达 | 次数 | 示例 |
|------|------|------|
| 开心 | 75 | 「我真的太开心了」「别提有多开心了」 |
| 幸福 | 64 | 「满满的幸福」「真的好幸福」 |
| 心里 | 38 | 「心里美滋滋的」「心里有点难过」 |
| 生气 | 30 | 「真的好生气」「想想就气」 |
| 难过 | 23 | 「心里一直很难过」 |
| 无奈 | 15 | 「我也真是无奈」 |

### ❌ 不能写

```
❌ 解释情绪：「我开心不是因为X，而是因为Y……」
❌ 分析情绪：「这种难过是源于多年来的……」
❌ 过度细化：「我感到既愤怒又悲哀，还掺杂着一丝释然」
```

**分界线**：说了「我很生气」→ ✅ 安全。说了「我为什么生气」→ ❌ 危险。

## 四、细节写法

### 写物理结果，不写诗

| AI式（❌）| 真人式（✅）| 原理 |
|-----------|-----------|------|
| 手有点抖 | 攥得起了褶子 | 写物理结果，不刻意传情绪 |
| 饭菜已经凉了 | 菜上面凝了一层白油 | 写客观状态，不暗示 |
| 心里一阵难过 | 眼泪就掉下来了 | 写发生的结果，不写内心 |

**自查口诀**：我写的这个细节，眼睛能直接看到吗？→ 能 → ✅。不能 → ❌。

## 五、句式规则

### 不用的句式
```
❌ 「不是X，是Y」结构（解释型）
❌ 「每个人都是……」结构（总结型）
❌ 过于工整的三段式排比
❌ 连续2句以上纯描写（无动作无对话）
```

### 安全的句式
```
✅ 「我本以为……没想到……」→ 偶尔用（230篇5次「没想到」）
✅ 「过了大概……」「后来有一次」→ 时间过渡
✅ 逗号串动作（「我进了屋，换了鞋，打开冰箱看了看」）
```

## 六、220-300 字短篇 vs 500-900 字标准篇

230 篇中两种体量并存，选哪种取决于素材：

| 体量 | 适合 | 结构 |
|------|------|------|
| 短篇（300-500字）| 单一个小事件，没有前因后果 | 一句话开头→事件→对话→停 |
| 标准篇（500-900字）| 有时间跨度的日常叙事 | 场景切入→过程展开→自然收尾 |

## 七、高频过渡词

```
自然时间：后来 过了几天 第二天 晚上 结果 没想到
对话衔接：然后他说 我就问他 她又说
动作串联：接着 于是 干脆 顺便
```

避用：然而、因此、此外、从那以后、慢慢地、我开始反思（230 篇中基本不出现）
